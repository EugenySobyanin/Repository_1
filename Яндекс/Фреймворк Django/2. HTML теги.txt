Теги:
	0.<!--какой-то текст--> (комментарии)
	1.<h1></h1>...<h6></h6> - заголовки
	2.<p></p> - абзац
	3.<a></a> - ссылка, между тегами - текст, по которому можно будет кликнуть, чтобы перейти по ссылке
		3.1. href - атрибут в котором указывают адрес ссылки
		3.2. target - атрибут который показывает как открыть страницу
			3.2.1 без атрибута или target="self" - открывает страницу в той же вкладке, заменив новой
			3.2.2 target="blank" - откроет в новой вкладке

	4.<img> - изобраджение, одиночный тег, с его помощью встраивают изображение (картинка может находиться как
	в директории проекта, так и на любом другом сервере.)
		4.1 src - атрибут в котором указывают адрес изображения 
	
	5.<b></b>(от англ.bold) - полужирное начертание
	6.<i></i>(от англ.italic) - выделить текст курсивом
	7.<br>(от англ. break, «разрыв») - разрыв строки
	8.<ul></ul> - маркированный список (неупорядоченный) (с точкам!)
		- вкладываются элементы, каждый элемент обрамляется тегами <li></li>
	9.<ol></ol> - нумерованный список
		- вкладываются элементы, каждый элемент обрамляется тегами <li></li>
	10.<div></div> (от англ. division, «отдел») - объединить в группу
	11.<hr> - видимая горизонтальная линия

Атрибуты тегов:
	- формат (имя="значение")

Структура HTML документа: 
	1. 1ый тег - <!DOCTYPE html> (указывает, что используется 5ая версия HTML)
	2. 2ой тег - <html lang="ru">...</html> (весь html код должен содержаться внутри этого тега)
		2.1 - необязательный атрибут lang - указывает на каком языке написано содержимое страницы
	3. <head>...</head> - содержить служебную информацию о странице, в нем указывают ссылки на дополнительные 
	служебные файлы, для правильного отображения страницы
		<meta> - теги о метаданных, метаданные - это данные о данных
		в нем указываю кодировку, например <meta charset="utf-8">
		в нем можно описать краткое описание, например: <meta name="description" content="Краткое описание страницы">
		<title>...</title> - заголовок, показывается на вкладке страницы

		favicon - маленькая картинка рядом с заголовком на вкладке, 
		подключается в теге <link> Пример: <link rel="icon" href="fav.ico" type="image">
		(подключают набор из разных иконок (формат .png или .gif) для разных размеров браузера
	
	
	4. <body>...</body> - в него обрамляют тело страницы, в нем находится все, что будет показано пользователю
	Разделы в теге <body>:
		1. <header></header> - шапка сайта, обычно размещают лого, навигацию
		2.<nav></nav> - в нем размещают навигацию, меню сайта. Этот тег может быть размещен в любой части сайта
		3.<main></main> - основная часть страницы, в ней могут содержаться более мелкие смысловые блоки:
			3.1 <section></section> - разделы, секции
			3.2 <article></article> - блоки, (от англ. - статья)
			3.3 <div></div> - структурные блоки
		4.<aside></aside> - это блок для доп. информации, косвенно связанной с основным содержанием
		5.<footer></footer> - подвал сайта
