CSS
- стили можно описать в отдельном файле, а в теге <head> дать ссылку на этот файл
Стили отдельно, HTML отдельно
	- парный тег <style>...</style> (помещается в блок <head>...</head>)
	Селекторы:
		- селектор по имени тега
			в описании указывается просто имя тега

		- селектора по классу: атрибут class
			-имя класса не обязано быть уникальным
			-можно записать несколько имен, разделяя их пробелом
			-в описании стилей перед именем класса ставится точка

		- селектор по индификатору id:
			-атрибут id
			-уникален в пределах страницы
			-в описании стилей перед именем id ставится #

	Комбинированные селекторы:
		- Есть разный синтаксис, примеры:
			- header * (все элементы внутри header)
			- p.negative (все элементы p c class="negative"
			- p .negative (все элементы с class="negative" вложенные в тег p
			- #anton p (все элементы p, вложенные в элементы с id="anton"


- другой способ, стили задаются внутри HTML с помощью атрибута style (применим к любому тегу)
(такой способо называют inline-стилями)
inline-стили
	- синтаксис: style="свойство: значение; другое_свойство: значение;"
