1. Подготовить данные для записи - словарь с данными
Структура:
        номер_договора: [сумма_за_год, сумма_за_расчетный_период, кол.во_дней, сумма_год_прописью, сумма_расчетная_прописью]

2. Запускаем цикл по словарю

3. Создаем новый файл (название - ключ словаря)

4. Копируем туда текст из example.txt

5. Редактируем текущий файл

    {№№} - номер договора
    {СУММ_Р} - сумма расчетного периода (без копеек)
    {ПРОПИСЬ_Р} - расчетный период прописью
    {РУБ_Р} - склонение Рубля для расчетного периода
    {КОПЕЙКИ} - копейки для расчетного периода
    {КОП_Р_СКЛ} - склонение копеек для расчетного периода
    {СУММ_Г} - сумма за год
    {ПРОПИСЬ_Г} - сумма за год прописью
    {РУБ_Г} - склонение рубля для года
    {КОЛВО_ДНЕЙ} - количество дней
