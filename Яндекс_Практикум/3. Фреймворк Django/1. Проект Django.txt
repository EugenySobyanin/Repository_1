Создание проекта:
	1.Создать корневую директорию проекта
	2.В корневой директории проекта создать виртуальное окружение - (python -m venv venv)
	3.Активировать виртуальное окружение из корневой директории - (source venv/Scripts/activate)
	4.Обновать пакетный менеджер pip - (python -m pip install --upgrade pip)
	5.В активированном виртуальном окружении развернуть фреймворк джанго - (pip install Django==3.2.16)
	6.Создать джанго проект - (django-admin startproject название_проекта)
	7.Запуск проекта - (python manage.py runserver) - запускается из директории где лежит файл manage.py
Создание приложения:
	1. - (python manage.py startapp название_приложения) - из директории где лежит файл manage.py
	2. зарегистрировать приложение в settings.py по формуле (директория.файл.класс)
