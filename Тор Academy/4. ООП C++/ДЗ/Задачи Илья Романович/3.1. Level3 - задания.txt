1. Программа должна сортировать массивы двумя способами. 
    Заполнение массивов выполнять автоматически, с помощью генератора случайных чисел rand. 
    Каждый алгоритм сортировки оформить как отдельную функцию, 
    которая возвращает значение характеризуемое трудоемкость алгоритма 
    (например, количество сравнений чисел или время, которое было потрачено на сортировку). 
    Выполнить сравнение алгоритмов на предмет эффективности.
        a).Сортировка выбором. Сначала выполняется поиск  минимального элемента в массиве, 
	    после чего сохраняется во временную переменную. Затем этот элемент удаляется в массиве, 
            а все последующие за ним элементы передвигаются на одну позицию влево. 
            После этого сохраненный элемент заносится  в  последнюю позицию, 
            которая освободилась после сдвига элементов влево.
	b).Шейкер-сортировка. Движение в прямом и обратном направлениях 
	    организовать с помощью одного цикла.

2. С клавиатуры вводится число N (типа int). Используя алгоритм «Решето Эратосфена», 
    необходимо найти все простые числа (т.е. делящиеся только на себя и на единицу) в интервале [0;N].

3. Разработать три класса, которые следует связать между собой, используя наследование:
    а).класс Product, который имеет три элемент-данных — имя, цена и вес товара 
	(базовый класс для всех классов);
    b).класс  Buy, содержащий данные о количестве покупаемого товара в штуках, 
	о цене за весь купленный товар и  о весе товара (производный класс для класса Product и 
	базовый класс для класса Check);
    c).класс Check, не содержащий никаких элемент-данных. Данный класс должен выводить на экран 
	информацию о товаре и о покупке ( производный класс для класса Buy);

    Для взаимодействия с данными классов разработать set- и get—методы. Все элемент-данные 
	классов объявлять как private
